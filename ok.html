The Most Common CSS Styles

Margin Explanation

Margin

The margin-top CSS property sets the margin area on the top of an element. A positive value places it farther from its neighbors, while a negative value places it closer.

Padding Explanation

Padding

The padding-top CSS property sets the height of the padding area on the top of an element.

Text Align

Text Align

The text-align CSS property sets the horizontal alignment of the content inside a block element or table-cell box. This means it works like vertical-align but in the horizontal direction.

Font Size

Font Size

The font-size CSS property sets the size of the font. Changing the font size also updates the sizes of the font size-relative units, such as em, ex, and so forth.

Background Color

Background Color

The background-color CSS property sets the background color of an element.

CSS Color

Color

The color CSS property sets the foreground color value of an element's text and text decorations. It also accepts the keyword currentcolor, which can be reused for other color-related properties like border-color.

CSS Width

Width

The width CSS property sets an element's width. By default, it defines the width of the content area. However, if box-sizing is set to border-box, it includes padding and border in the total width.

CSS Line Height

Line Height

The line-height CSS property sets the amount of space used for lines of text. It helps improve readability by controlling the spacing between baselines of consecutive lines. This is especially useful when styling paragraphs or lists.
